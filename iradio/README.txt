Installation:

- Unzip all the files in the your /modules/ folder.
- Chmod the cache folder & files to 777 & 666
- Activate module in your admin menu. 
- Set user access rights
- Add/edit/remove radiostations in the admin panel. 

<note>
If you  want to add this module to main menu, just change "$modversion['hasMain']   = 0" like this!
$modversion['hasMain']      = 1;  
(In xoops_version.php)

Enjoy !!!!
