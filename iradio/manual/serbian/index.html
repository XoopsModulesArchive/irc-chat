iRadio Module Manual

General

This module is a port of a PHP-Nuke module written by Rene Hart. It provides a block that will play streamed radio from one of a list of radio stations chosen by you.

The E-Xoops version has been modified to include configuration options accessible through the System Admin menu.

Install

Should be completely straightforward. Just unzip into your site folder and remember to chmod the cache folder and files to 777 & 666.

In language files other than English some entries will need correction!

Configuration

There is a short list of Internet radio providers held in a database. You will need to edit this list to include the stations you want to see on your site. See The MIT Radio Locator for a useful list.
